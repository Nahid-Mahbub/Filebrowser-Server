# FileBrowser Server

A lightweight Bash-based utility to manage a **FileBrowser server** on Linux.  
It provides simple scripts to install requirements, start, stop, and monitor a local network file server.

## ✨ Features

- 🚀 One-command FileBrowser setup
- 📦 Automatic dependency installation
- 🌐 Local & LAN access support
- 🔄 Automatically handles existing server processes
- 📝 Server log management
- 📊 Server status monitoring
- 💻 Lightweight Bash scripts
